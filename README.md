# ad-variations

Generate diverse Facebook/Meta ad creative variations using HAF framework. Use when the user asks for ad variations, creative testing assets, hook-angle-format combinations, Meta ad creatives, or Facebook ad images for a product or brand.

## Install

Paste this in your terminal:

```bash
mkdir -p ~/.claude/skills/ad-variations && curl -sL "https://raw.githubusercontent.com/jangess-skills/ad-variations/main/SKILL.md" -o ~/.claude/skills/ad-variations/SKILL.md
```

Then use `/ad-variations` in Claude Code.

## Uninstall

```bash
rm -rf ~/.claude/skills/ad-variations
```
